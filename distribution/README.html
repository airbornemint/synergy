Synergy 4.0



System requirements


Synergy 4.0 is a ground-up rewrite of the older 3.0 codebase especially for Snow Leopard. It will not run on older versions of Mac OS X. (The previous version, Synergy 3.4, runs on all versions of Mac OS X from Jaguar through to Leopard, and it can be downloaded from:


http://wincent.com/a/products/synergy-classic/download/tiger/



Licenses


If you purchased your Synergy license within the last three years (on or after 5 September 2009) your license will work with Synergy 4.0 (and with subsequent updates such as 4.1, 4.2 and so on).


If you purchased your license more than three years ago then it will not work with Synergy 4.0. You can either purchase a new license at:


https://secure.wincent.com/a/products/synergy-classic/purchase/


Or continue to use a version of Synergy prior to 4.0.



Installation


To install, drag the Synergy Preferences application to a location of your choosing. If you want it to be accessible from all user accounts on your system then the "Applications" folder on your startup volume is a good location.


If you have an old version of Synergy on your system the new version will detect it and offer to clean it up for you.



What happened to the installer?


Old versions of Synergy were distributed as a System Preferences pane, which meant that they had to be installed into a specific location in order for to work. Now that it is a standalone application it will work from any location and doesn't require an installer.


The underlying cause of this change is that Synergy 4.0 takes advantage of a new Leopard technology called Garbage Collection, and this is incompatible with the System Preferences application, thus requiring that Synergy adopt a new format and cease to be a System Preferences pane.



Uninstallation


To uninstall, drag the Synergy Preferences application to the Trash.



Starting Synergy


To start Synergy, double-click the Synergy Preferences application and click the Start button in the General tab.



Stopping Synergy


To stop Synergy, click the Stop button in the General tab of the Synergy Preferences.
